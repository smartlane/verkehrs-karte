Informationen

Die Website "Mobil Bei Uns" sammelt Verkehrsbehinderungen aus verschiedenen offenen Quellen und stellt sie gemeinsam auf einer Karte dar.

Die Nutzung

Alle Verkehrsbehinderungen können auf der Karte eingesehen werden. Über den Menüpunkt "Region / Stadt auswählen" lassen sich bestimmte Regionen zentrieren.

Alle Daten können auch zur weiteren Verwendung via JSON-Schnittstelle abgerufen werden. Auf einer separaten Seite gibt es Details zu der API.

Neue Datenquellen

Um eine neue Stadt hier aufzunehmen, werden die Daten als OpenData benötigt. OpenData bedeutet: Die Daten müssen als JSON-, XML- oder CSV-Datei vorliegen. Sie müssen eine freie Lizenz haben (bestenfalls CC-0). Und für den hier genutzten Abruf müssen die Daten immer unter derselben URL verfügbar sein.

Als Bürger einer Stadt können Sie einfach eine E-Mail an Ihre Verwaltung schreiben, um dies anzustoßen. Als Mitarbeiter einer Stadt können Sie anfangen, eine OpenData Strategie für Ihre Stadt zu entwickeln - die OKF Deutschland hilft Ihnen dabei gerne.

Elektromobilität?

"Mobil Bei Uns" ist ein Projekt des ruhrmobil-E e.V., eines Vereins für Elektromobilität und regenerativer Energie. Wir glauben, dass die Mobilität der Zukunft nur vernetzt funktionieren kann. Für eine vernetzte Mobilität sind Daten der wichtigste Rohstoff: z.B. eben Verkehrsbehinderungen.

Die Zukunft

Wir wollen nicht nicht nur bei Baustellen bleiben, sondern auch weitere Datenquellen integrieren. Dies könnten z.B. sein: ÖPNV-Verspätungen, defekte Aufzuüge, Staus, ... etc.
